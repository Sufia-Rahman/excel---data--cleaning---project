# Excel Data Cleaning and Analysis Project

## Overview
This project demonstrates how raw, messy sales data was cleaned, formatted, and analyzed using Microsoft Excel. The goal was to convert unstructured data into a clean, organized, and analysis-ready dataset.

## Data Cleaning Tasks Performed
- Removed duplicate records  
- Fixed inconsistent date formats  
- Cleaned and standardized email text (uppercase/lowercase issues)  
- Handled missing values  
- Formatted columns properly  
- Organized raw data into a structured table  

## Data Analysis (Pivot Tables)
After cleaning the data, I created multiple pivot tables to analyze:

- Sales by product line  
- Sales by country  
- Sales by year and month  
- Customer-wise sales insights  

These pivot tables help in understanding trends and making data-driven decisions.

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Basic Excel formulas  

## Result
Transformed messy and inconsistent data into a clean, structured dataset and generated meaningful insights using pivot tables.
